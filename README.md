# First Conviction

Public multi-page website for First Conviction.

## Pages

- `/` Home
- `/pathway` Pathway
- `/constitution` Constitution
- `/declaration` Declaration

## Stack

- Static HTML
- Shared CSS
- Small vanilla JavaScript enhancement layer
- Vercel-ready static deployment

## Local Preview

Because this is a static site, it can be opened directly in a browser or served locally with any static file server.

## Deployment

### GitHub

1. Initialize or use the existing git repository.
2. Push to a GitHub repository.

### Vercel

1. Import the GitHub repository into Vercel.
2. Use the repository root as the project root.
3. No build command is required.
4. Output directory should remain unset.

The included `vercel.json` enables clean public URLs like `/pathway` instead of `/pathway.html`.
